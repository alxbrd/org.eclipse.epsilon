Getting help

If you have a question that is not answered in the documentation, please send a message to the eclipse.epsilon newsgroup. Under normal circumstances and within working hours/days (9-5 GMT), you should get a response to your question within a few minutes. Instructions for sending messages to Eclipse newsgroups are available here.

Reporting bugs

If you believe you've found a bug in Epsilon, please consider submitting a bug report to the Eclipse Bugzilla and we'll try to fix it as soon as possible.
