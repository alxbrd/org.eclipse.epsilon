Epsilon Interim Update Site

This site is designed to be used through the Eclipse update manager, which is accessible within Eclipse under Help->Software Updates. For instructions on how to install Epsilon using this site, please refer to the installation guide.

Caution: This is an interim update site, which means that the version available here contains the latest features of Epsilon but some of them may not have been properly tested yet. For the latest stable version of Epsilon, please use the main update site instead.
